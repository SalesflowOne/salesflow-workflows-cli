Privacy Policy

Salesflow Workflows Token Grabber — a Chrome extension by Salesflow One.

Last updated: June 25, 2026

What this extension does

This extension reads your Firebase authentication refresh token from browser-local storage (IndexedDB) on GoHighLevel whitelabel pages you have open, such as app.salesflow.one. The token is used to configure the Salesflow Workflows CLI so you can create and update workflows from the terminal.

Data collection

  • Firebase refresh token — read only when you click “Grab Refresh Token” on an allowed GHL page.
  • Active tab URL — checked locally to confirm you are on an allowed domain before reading storage.

What we do not do

  • We do not send your token or any data to Salesflow One or any third-party server.
  • We do not track browsing history beyond the active tab check described above.
  • We do not sell or share user data.

Where your data goes

The refresh token stays on your device. When you click “Copy to Clipboard”, it is copied to your system clipboard so you can paste it into your local .env file for the CLI. You control that file on your machine.

Permissions

  • activeTab — access the current tab only when you open the extension popup.
  • scripting — run a short script in the page context to read IndexedDB.
  • clipboardWrite — copy the token when you click “Copy to Clipboard”.
  • host_permissions — limited to GoHighLevel / Salesflow app domains where Firebase storage exists.
