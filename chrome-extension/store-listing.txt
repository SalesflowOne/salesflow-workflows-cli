ITEM NAME
Salesflow Workflows Token Grabber

SUMMARY (132 chars max)
Grab your GoHighLevel Firebase token on app.salesflow.one for the Salesflow Workflows CLI. Local only — nothing sent to servers.

DETAILED DESCRIPTION
Salesflow Workflows Token Grabber helps Salesflow One customers configure the Salesflow Workflows CLI for GoHighLevel workflow building.

The public GoHighLevel API cannot create workflows. The Salesflow Workflows CLI uses GHL's internal API, which requires a Firebase refresh token from your logged-in browser session.

How it works:
1. Log in to https://app.salesflow.one (or your GHL whitelabel domain).
2. Click the extension icon.
3. Click "Grab Refresh Token".
4. Click "Copy to Clipboard".
5. Paste into your CLI .env as GHL_FIREBASE_REFRESH_TOKEN.

Privacy:
- Reads Firebase auth data from browser IndexedDB only when you click the button.
- Does not transmit data to any server.
- Token is copied to your clipboard only when you choose to copy it.
